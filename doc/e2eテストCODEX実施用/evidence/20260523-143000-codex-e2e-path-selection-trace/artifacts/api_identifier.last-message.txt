**今回守る制約**
- 管理者ログインでは `MFA_REQUIRED_ON_ADMIN_LOGIN` を必須とし、MFA 成功まで管理画面セッションを発行しない。根拠: Source Specs `docs/spec/login.md#0002-admin-login-mfa`、Purpose。
- 管理者ログインの成功時と失敗時には `AUDIT_EVENT_REQUIRED` を記録する。記録項目は `actor_id`, `source_ip`, `result`, `mfa_result`, `occurred_at`。根拠: Source Specs `docs/spec/login.md#0003-audit-event-required`。
- `AUDIT_EVENT_REQUIRED` の保存先は `audit_log_stream` とする。根拠: Source Specs `docs/spec/audit.md#0002-audit-storage-boundary`。
- MFA 成功前に管理画面セッションを発行しない制約を優先する。根拠: Conflict Review Item `conflict-login-audit-human-resolution`。
