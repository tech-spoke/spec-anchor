### 対処手順

**前回の collection はこの verify 失敗によって drop されていません。** 既存データは保持されたままです。

不整合を解消するには、以下のコマンドで section collection を作り直してください:

```bash
spec-anchor core --rebuild
```

または同等のスキル呼び出し: `/spec-core --rebuild`

これにより Retrieval Index が全セクションを対象に再構築され、`docs/spec/audit.md#0003-logout-audit` の欠落も解消されます。
