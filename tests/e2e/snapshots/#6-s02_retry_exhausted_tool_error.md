# #6-s02 構造化失敗 → リトライ → 再失敗 → ⑥ ツール側エラー

Agent が答案を組み立てたが、1 回目も再構成後の 2 回目も形式不備で停止した。⑥ ツール側のエラーへ落とし、最後に送った答案と「期待された形式との差分」を利用者に併記する。内部 field 名は使わず、人間語で差分を説明する。

---

■ ツール側でエラーが発生しました

  回答候補の形式を 2 回そろえられませんでした。これは利用者の操作では解消できない可能性があります。

  期待された形式との差分:
    - 「課題プロンプトへの回答または修正案」のセクションが空のままでした (回答本文が必要です)。

  最後に送った回答候補:
    今回守る制約:
      - 連続ログイン失敗がしきい値を超えたらアカウントを一時ロックする。
    今回扱う修正候補または検討対象:
      - 失敗回数のカウント単位を確定する。
    競合 / 不確実性 / 人間レビューが必要な点:
      - ロック時間の具体値は資料に明記なし。
    課題プロンプトへの回答または修正案:
      - (空のまま)

  次の操作:
    本ツールの開発元へ、上記の差分と実行したコマンドを添えて報告してください。
