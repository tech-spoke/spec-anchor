# #2-s01 ① 初期設定が完了していない (/spec-core)

利用者が `/spec-core` を、まだ初期化していないプロジェクトで実行したときの Agent 整形応答。

---

■ 初期設定が完了していません

  プロジェクトの設定ファイル (.spec-anchor/config.toml) が見つかりませんでした。

  次の操作:
    spec-anchor-setup-project --target /path/to/project を実行して初期化してください。
    Purpose / Core Concept ファイル (docs/core/purpose.md 等) が未作成なら、人間が作成してください。
