# #2-s05 ⑥ ツール側のエラー (/spec-core)

利用者が `/spec-core` を呼び、保持物更新の途中で想定外のエラーが起きたときの Agent 整形応答。

---

■ ツール側でエラーが発生しました

  処理中に想定外のエラーが発生しました。これは利用者の操作では解消できない可能性があります。

  エラー内容: Source Retrieval Index の更新に失敗しました。

  次の操作:
    /spec-core --rebuild で再構築を試してください。解消しない場合は、本ツールの開発元へ上記エラー内容と実行したコマンドを添えて報告してください。
